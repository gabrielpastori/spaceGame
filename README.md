# Instruções para execução:
No projeto, acesse:  
`cd spacegame/`  
Então, execute:  
`python3 main.py`  

# Fases:
O jogo é composto por 3 fases, em cada uma delas é atribuído um determinado fundo, resistência progressiva aos asteroides e armas diferentes.

# Comandos:
* Ctrl: Atira
* Setas: Movimento da nave
* C: Inicia o Jogo
* Q: Sai do jogo
* P: Pausa o jogo
